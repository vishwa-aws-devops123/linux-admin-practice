# Linux Administration Interview Questions

## What is Linux?

Linux is an open-source operating system widely used for servers, cloud environments, and enterprise applications.

## What is SSH?

SSH stands for Secure Shell. It is used to securely connect to remote Linux servers.

## What is systemctl?

systemctl is used to manage services in Linux such as start, stop, restart, enable, disable, and status checks.

## Difference between df and du?

* df shows filesystem disk usage.
* du shows disk usage of files and directories.

## What is grep?

grep is used to search for text patterns inside files.

## What is find?

find is used to search for files and directories based on various criteria.

## What are file permissions?

File permissions determine who can read, write, and execute a file.

## What is chmod?

chmod is used to change file permissions.

## What is chown?

chown is used to change file ownership.

## What is a Linux service?

A service is a background process managed by systemd and systemctl.
