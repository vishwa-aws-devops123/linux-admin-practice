# Networking Commands

## Check IP Address

```bash
ip addr
```

## Check Connectivity

```bash
ping google.com
```

## Display Routing Table

```bash
ip route
```

## Check Listening Ports

```bash
ss -tulnp
```

## DNS Lookup

```bash
nslookup google.com
```

## Check Network Interfaces

```bash
ifconfig
```

## Trace Network Path

```bash
traceroute google.com
```
